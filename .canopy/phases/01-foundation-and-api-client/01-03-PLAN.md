<task type="auto">
  <name>Task 1: Create Cortex API client with full instrumentation</name>
  <files>src/api/client.ts</files>
  <action>
    Create the API client module at `src/api/client.ts`:

    1. Import OpenAI from 'openai', performance from 'perf_hooks', and project modules (eventBus, logger, classifyError, types).

    2. Create the OpenAI client instance:
       ```typescript
       const client = new OpenAI({
         apiKey: config.apiKey,
         baseURL: config.baseUrl,  // 'https://cortex.nfinitmonkeys.com/v1'
         maxRetries: 0,            // CRITICAL: disable auto-retry so we classify errors ourselves
         timeout: 30_000,          // 30s timeout
       });
       ```
       Note: The client should accept config as a parameter (from validateConfig), not read process.env directly. This keeps it testable.

    3. Create `makeRequest` function:
       ```typescript
       export async function makeRequest(
         messages: OpenAI.ChatCompletionMessageParam[],
         config: AppConfig
       ): Promise<CortexResponse>
       ```

       Implementation:
       a. Emit `api:request` event with `{ model: config.model, messageCount: messages.length, timestamp: Date.now() }`
       b. Capture `startMs = performance.now()` IMMEDIATELY before the API call
       c. Call `client.chat.completions.create({ model: config.model, messages })`
       d. Capture `latencyMs = performance.now() - startMs` IMMEDIATELY after (before any processing)
       e. Round latencyMs to 2 decimal places: `Math.round(latencyMs * 100) / 100`
       f. Extract `usage.prompt_tokens` and `usage.completion_tokens` (default to 0 if missing)
       g. Extract `choices[0].message.content` (default to empty string if missing)
       h. Emit `api:response` event with latency, tokens, model, content
       i. Log with pino: `logger.info({ event: 'api_response', latencyMs, promptTokens, completionTokens, model })`
       j. Return `CortexResponse` object

       Error handling (catch block):
       a. Capture `latencyMs` even on error
       b. Call `classifyError(error)` from Plan 02's errors module
       c. Emit `api:error` event with classified error + latencyMs
       d. Log with pino: `logger.warn({ event: 'api_error', errorType: classified.type, statusCode, retryAfterMs, latencyMs })`
       e. Throw the `ClassifiedError` (not the original error) — caller decides retry strategy

    4. Export a `createCortexClient` function that takes `AppConfig` and returns an object with `makeRequest` bound to that config. This allows the entry point to create the client once after config validation.

    5. Run `npx tsc --noEmit` to verify compilation.
  </action>

  </verify>
  <done>API client compiles, uses performance.now() for latency, emits typed events, classifies errors, logs with pino — all wired to Plan 01 types and Plan 02 error classification</done>
</task>

       a. Call `validateConfig()` — this will exit with clear error if API key missing
       b. Create cortex client: `const cortex = createCortexClient(config)`
       c. Set up event bus listeners for demonstration:
          - `eventBus.on('api:request', (evt) => logger.debug({ event: 'bus:api_request', ...evt }))`
          - `eventBus.on('api:response', (evt) => logger.debug({ event: 'bus:api_response', ...evt }))`
          - `eventBus.on('api:error', (evt) => logger.debug({ event: 'bus:api_error', ...evt }))`
       d. Log startup: `logger.info('StressCortex starting — making single test call to Cortex API')`
       e. Make a single test call:
          ```typescript
          const result = await cortex.makeRequest([
            { role: 'system', content: 'You are a helpful assistant. Respond briefly.' },
            { role: 'user', content: 'Say "Hello from StressCortex" and nothing else.' }
          ]);
          ```
       f. Log the result:
          ```typescript
          logger.info({
            event: 'test_call_complete',
            content: result.content,
            latencyMs: result.latencyMs,
            promptTokens: result.promptTokens,
            completionTokens: result.completionTokens,
            model: result.model,
          }, 'Single-turn test call succeeded');
          ```
       g. Print human-readable summary to console:
          ```
          ✓ Cortex API call succeeded

            Model:    {model}
          ```

    3. Call `main()` with `.catch()`:
       - If error is a ClassifiedError, log it and exit with code 1
       - If unknown error, log it with logger.fatal and exit with code 1
